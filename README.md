# Escala Equipe 9132

Aplicativo web simples para visualizar a escala por nome, dia ou posto.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie todos os arquivos deste projeto para o repositório.
3. Vá em Settings > Pages.
4. Em Source, escolha Deploy from a branch.
5. Escolha branch main e pasta / root.
6. Salve e aguarde o link do GitHub Pages.

## Como editar a escala

Abra o arquivo `dados.js` e altere os dados da constante `ESCALA`.

Cada pessoa possui 31 posições dentro de `dias`, representando os dias 1 a 31.
